# 📱 WEBFIN - Sistema de Gestão de Bolso

Um aplicativo web progressivo (PWA) de código aberto, focado em simplicidade, privacidade e controle financeiro para pequenos empreendedores e autônomos. Feito para rodar offline direto no celular, substituindo o caderninho de anotações.

## 🎯 O Propósito (A História por trás do código)
Este projeto começou como uma ferramenta de uso pessoal. Porém, tudo mudou em uma sexta-feira, após uma conversa com uma vendedora de bolo de pote. O método de controle financeiro dela era brutalmente simples e perigoso: *"Vendi todo o meu cooler, tenho 240 reais"*. Ela separava metade para si e metade para os insumos. Não havia cálculo de margem, provisionamento, investimento ou sequer o custo do almoço dela embutido no dia de trabalho. 

O WEBFIN nasceu para pessoas como ela. É para quem não tem dinheiro para pagar por um sistema de ERP complexo, não tem tempo para aprender softwares difíceis e, principalmente, **não quer ter seus dados roubados ou ser bombardeado por propagandas**.

## 🛡️ Filosofia do Sistema
* **100% Gratuito e Sem Anúncios:** A ferramenta é para ajudar, não para lucrar em cima de quem já está na batalha diária.
* **Privacidade Absoluta (Offline First):** O sistema não tem um servidor de banco de dados na nuvem. Tudo que você digita fica salvo **exclusivamente na memória do seu celular** (via IndexedDB). Ninguém além de você tem acesso aos seus números.
* **Focado no Mobile:** A interface não é um site de computador espremido. É desenhada puramente para a usabilidade de um celular.

## ⚙️ Funcionalidades Atuais
