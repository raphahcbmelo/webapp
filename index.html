📱 INSTALAÇÃO DO SISTEMA

Realize o backup dos seus dados antes de criar um novo atalho na tela inicial.

SISTEMA ANDROID: Navegador Chrome > Menu (⋮) > Adicionar à tela inicial.

SISTEMA iOS: Navegador Safari > Compartilhar > Adicionar à Tela de Início.

ℹ️ GUIA DE USO

👤 EU: Central de dados pessoais. Cadastre contas bancárias para criar atalhos rápidos de cópia de PIX.

💲 REGISTROS: Formulário de fluxo de caixa. Insira receitas e despesas. Permite configuração de repetição automática.

📓 DIÁRIO: Livro Razão. Visualize a linha do tempo financeira. Possui controles para edição (✏️) e exclusão (🗑️).

🏢 MÓDULOS CORPORATIVOS: Ferramentas para gestão de CNPJ, inventário e controle de clientes.

📄 RELATÓRIOS: Cruzamento de dados para geração de tabelas auditáveis. Otimizado para impressão direta em A4.
